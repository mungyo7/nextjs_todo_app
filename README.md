# 📝 Next.js Todo 애플리케이션

## ✨ 주요 기능

- ⚡ 실시간 할 일 추가/수정/삭제
- 🔄 데이터베이스 연동 (MongoDB)

## 🛠 기술 스택

- **Frontend**: Next.js
- **Backend**: Next.js API Routes
- **Database**: MongoDB
- **Styling**: Tailwind CSS

## 🚀 시작하기

root 폴더 .env.local 파일 생성

MONGODB_URI=mongodb+srv://<username>:<password>@<cluster>.mongodb.net/todo?retryWrites=true&w=majority&appName=todo
